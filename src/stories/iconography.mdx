# Iconografia

Aqui estão alguns exemplos de ícones disponíveis na biblioteca Fluent UI.

## Ícones Importados

import { Add16Regular } from '@fluentui/react-icons';

### Exemplo de Ícones

- **Add16Regular**: <Add16Regular style={{ fontSize: '24px', color: '#0078d4' }} />

## Como Usar

Para usar esses ícones em seu projeto, basta importá-los do pacote `@fluentui/react-icons` e renderizá-los como componentes React. Por exemplo:

```jsx
import { Add16Regular } from '@fluentui/react-icons';

<Button icon={<Add16Regular style={{ fontSize: '24px'}} />}></Button>
